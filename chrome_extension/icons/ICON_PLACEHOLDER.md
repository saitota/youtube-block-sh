# Icon Placeholder

アイコンファイルが必要です：

## 必要なファイル
- `icon-toolbar.png` (16x16) - ツールバー用
- `icon-extension.png` (48x48) - 拡張機能一覧用  
- `icon-store.png` (128x128) - ストア用

## デザインコンセプト
- 赤い円に白い✖マーク（ブロック記号）
- YouTubeのような配色（赤・白）
- シンプルで認識しやすいデザイン

## 代替案
現在のところ、以下の文字列をアイコン代わりに使用可能：

```
🚫 - ブロック記号
⛔ - 禁止記号
🛑 - 停止記号
```

実際のPNGファイルが必要な場合は、上記のデザインでSVGからPNGに変換して作成してください。
